## 决策记录

| 决策 | 选项 | 理由 |
|---|---|---|
| LLM 调用位置 | **Rust 后端** | 保持 Tauri command 合约不变 |
| LLM 配置来源 | **.env 静态 + 设置面板运行时覆盖** | .env 设默认值，运行时可通过面板覆盖 API key/endpoint/model |
| 右键菜单 | **转型为宠物管理中心** | LLM 自主决策动画，用户无需手动切动画 |
| 窗口尺寸 | **动态展开/收缩** | 平时=精灵尺寸，菜单/overlay 打开时自动扩展 |
| 代理 API 适配 | **全手动输入** | 适配 OpenAI/DeepSeek/Ollama/vLLM 等任何兼容 API |
| 设定持久化 | **localStorage** | Tauri 运行时无法写 .env，localStorage 够用 |
